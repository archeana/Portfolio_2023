First Entry

July 1, 2023

This is my very first blog entry.

  1. Extension I started doing this website using the tutorial from Code makery. The first thing I learned was how to add an extension to the Visual Studio code Editor. The "Live server" extension. I have been using Visual Studio Code for 2 months and did not know that existed. Now I can see it live, instead of running and opening in the browser and accidentally running the CSS script instead of the HTML.
  2. Publish website. I learn how to publish my website on Netlify. Wow is so easy to "publish" just drop the folder. It is definitely easier than Github, but do I have to drop the folder all the time? We will see when I finish.
  3. CSS CSS is how to make HTML look pretty. I learn a lot of basic things that I did not know. Using the DevTools is so much cool. I can see if something is not working and I don't need to delete in VScode I can just click and unclick and see what is the problem.
  4. Pages I actually did not know how this worked. I did a course ~2 hours a week to learn to do a website and in the end, we had to do a website in our group. When I was doing this course, we divided the job by 4 so each person had ca.3 pages to do and we made one html and CSS for each. It was such a headache afterwards to add all to one website. But now I read this post on how to create pages and makes so much sense. Mindblowing. Now I created all the html and folders for this website.
  5. Format document in VScode I just discover the "format document" so useful now my code looks so neat ^-^. I just started watching youtube videos on Flexbox, Bootstrap and ways to do layouts. I try a couple of them and then got stuck.
  6. Favicon I learn to add a favicon. I use Canva to do the icon and then add it to the website. Now looks cool. I started doing the navbar using Flexbox. I used this youtube to guide me Youtube video. Some parts did not work so I need to figure it out. Like the images go under the navbar. ?? I wanted to make it responsive to all devices. But in the end, it was too complicated and it did not work.

All for today!
